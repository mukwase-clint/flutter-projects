# 🛒 Shopping List App

A Flutter application that allows users to create and manage a shopping list by adding items, selecting categories, and setting quantities for each item.

---

## ✨ Features

- Add shopping items
- Select item categories
- Set item quantities
- View all added items
- Simple and clean user interface
- Delete items

---

## 🛠️ Built With

- Flutter
- Dart
